# Запуск приложения через Android Studio

## Быстрый старт:

1. **Откройте проект** в Android Studio:
   - File → Open
   - Выберите папку: `client/android`

2. **Дождитесь синхронизации Gradle**
   - Android Studio автоматически начнет синхронизацию
   - Внизу будет индикатор прогресса

3. **Подключите телефон** через USB:
   - Включите "Отладка по USB" на телефоне
   - Настройки → О телефоне → Нажмите 7 раз на "Номер сборки"
   - Настройки → Для разработчиков → Отладка по USB

4. **Запустите приложение**:
   - Нажмите зеленую кнопку **▶ Run 'app'** (или Shift+F10)
   - Или: Run → Run 'app'
   - Выберите ваше устройство из списка

5. **Приложение установится и запустится** на телефоне

## Если возникают проблемы:

### Ошибка "Gradle sync failed":
1. File → Invalidate Caches → Invalidate and Restart
2. Дождитесь перезапуска Android Studio
3. Попробуйте снова

### Ошибка "Device not found":
1. Проверьте, что телефон подключен: `adb devices`
2. Убедитесь, что включена отладка по USB
3. На телефоне появится запрос "Разрешить отладку?" → Нажмите "Разрешить"

### Ошибка при установке:
1. Удалите старое приложение с телефона
2. Build → Clean Project
3. Build → Rebuild Project
4. Запустите снова

### Приложение не запускается:
1. Откройте **Logcat** (внизу экрана)
2. Выберите ваше устройство и приложение
3. Ищите ошибки (красным цветом)
4. Скопируйте ошибку для анализа

## Полезные команды:

**Проверка подключенных устройств:**
```bash
adb devices
```

**Очистка проекта:**
```bash
cd android
./gradlew clean
```

**Пересборка:**
```bash
./gradlew assembleDebug
```

## Что происходит при запуске:

1. Android Studio собирает проект (если нужно)
2. Устанавливает приложение на телефон
3. Запускает приложение
4. Можно видеть логи в Logcat в реальном времени

## Отладка:

- **Logcat** - все логи приложения
- **Breakpoints** - точки останова для отладки
- **Network Inspector** - просмотр сетевых запросов (View → Tool Windows → Network Inspector)

## Горячие клавиши:

- **Shift + F10** - Запустить приложение
- **Shift + F9** - Запустить с отладкой
- **Ctrl + F9** - Собрать проект
- **Alt + 6** - Открыть Logcat
