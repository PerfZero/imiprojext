# Исправление проблемы с установкой APK

## Проблема:
- "Invalid file" при установке APK
- "package com.imi.club not found"
- Ошибки AppOps

## Решение:

### 1. Удалите старое приложение с телефона:

**Вариант А: Через настройки телефона**
1. Настройки → Приложения
2. Найдите "IMI CLUB"
3. Нажмите "Удалить"

**Вариант Б: Через ADB (если телефон подключен)**
```bash
adb uninstall com.imi.club
```

### 2. Очистите проект в Android Studio:

1. В Android Studio: **Build → Clean Project**
2. Затем: **Build → Rebuild Project**

### 3. Пересоберите APK:

1. **Build → Build Bundle(s) / APK(s) → Build APK(s)**
2. Дождитесь завершения сборки

### 4. Установите новый APK:

**Вариант А: Через Android Studio**
1. Подключите телефон через USB
2. Включите "Отладка по USB" на телефоне
3. В Android Studio: **Run → Run 'app'**
4. Выберите ваше устройство

**Вариант Б: Вручную**
1. Скопируйте APK на телефон
2. Откройте файловый менеджер на телефоне
3. Найдите APK файл
4. Нажмите на него для установки
5. Разрешите установку из неизвестных источников (если нужно)

### 5. Если все еще не работает:

**Проверьте подпись APK:**
```bash
cd android/app/build/outputs/apk/debug
jarsigner -verify -verbose -certs app-debug.apk
```

**Пересоберите с очисткой:**
```bash
cd android
./gradlew clean
./gradlew assembleDebug
```

**Проверьте размер APK:**
APK должен быть больше 1MB. Если меньше - сборка не завершилась.

### 6. Альтернативный способ установки:

Если APK не устанавливается, попробуйте через ADB:
```bash
adb install -r android/app/build/outputs/apk/debug/app-debug.apk
```

Флаг `-r` переустановит приложение, если оно уже установлено.

## Что было исправлено в коде:

1. ✅ Добавлен `package="com.imi.club"` в AndroidManifest.xml
2. ✅ Увеличен `versionCode` с 1 до 2
3. ✅ Обновлен `versionName` до "1.0.1"

Эти изменения помогут избежать конфликтов с предыдущими установками.
