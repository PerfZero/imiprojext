# Простая инструкция: Как создать APK файл

## Шаг 1: Установите Android Studio

1. Скачайте Android Studio с официального сайта: https://developer.android.com/studio
2. Установите его на ваш компьютер
3. При первом запуске Android Studio установит все необходимые компоненты (это займет время)

## Шаг 2: Откройте проект

1. Запустите Android Studio
2. Выберите **"Open"** (Открыть)
3. Найдите и выберите папку: `client/android`
4. Дождитесь, пока Android Studio загрузит проект (может занять несколько минут)

## Шаг 3: Соберите APK

1. В верхнем меню Android Studio выберите **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
2. Дождитесь завершения сборки (внизу будет показан прогресс)
3. Когда сборка завершится, появится уведомление
4. Нажмите на ссылку **"locate"** в уведомлении, чтобы открыть папку с APK

## Где найти готовый APK?

APK файл будет находиться здесь:
```
client/android/app/build/outputs/apk/debug/app-debug.apk
```

Этот файл можно установить на любой Android телефон!

## Что делать, если что-то не работает?

1. Убедитесь, что Android Studio полностью установлен
2. Дождитесь завершения синхронизации Gradle (внизу экрана)
3. Если есть ошибки, прочитайте их и попробуйте исправить

## Альтернативный способ (через командную строку)

Если у вас установлен Gradle, можно собрать APK через терминал:

```bash
cd client/android
./gradlew assembleDebug
```

APK будет в той же папке: `app/build/outputs/apk/debug/app-debug.apk`
