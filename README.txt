IMI CLUB dev v1

Стек
Фронт: Vite, Vue(TS),PWA, bootstrap, шаблон 8mobile_bootstrap_4_mobile
Бэк: NodeJS(TS, expressjs, better-auth, drizzle, swagger), Postgres, Redis


ДЛЯ РАЗРАБОТЧИКА / БЫСТРЫЙ СТАРТ

1. Запустить сервер(папка /server)
cd server
npm install
npm run dev

2. Запустить клиент(папка /client)
cd client
npm install
npm run dev
Открыть в браузере http://localhost:5173/
